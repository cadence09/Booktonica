# Booktonica
This is a book app to allow users to make comments on individual book.

# Code style
The code formats in ESLint 

# Tech/framework used

## Build with
  * React
  * Express
  
# Installation
  * Clone my repo <br/>
     - git clone "my repo"
  * Create database <br/>
     - createdb booktonica
     - psql booktonica < booktonica.sql
  * Run the app <br/>
     - cd express-api
     - npm install
     - npm start
     - cd ../react-app
     - npm install
     - npm start
